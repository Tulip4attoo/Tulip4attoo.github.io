Local development (Ubuntu)
--------------------------
- Install Ruby and build tools: `sudo apt update && sudo apt install -y ruby-full build-essential zlib1g-dev`
- Install Bundler: `gem install bundler`
- Install dependencies from the repo root: `bundle install`
- Serve locally with auto-reload: `bundle exec jekyll serve --livereload`
- Open http://localhost:4000/ to view the site
- To build without serving: `bundle exec jekyll build`
